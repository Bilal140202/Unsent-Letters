# Unsent-Letters - AI Build Blueprint

This repository contains a full-stack web application for anonymously writing and sharing unsent letters.

## Goal
To maintain a safe, performant platform with clear separation of concerns between the frontend UI and the backend data handling.

## AI Guidelines
